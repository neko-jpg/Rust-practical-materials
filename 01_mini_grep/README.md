# Level 1: Mini Grep

## 課題
指定されたファイルから、特定の文字列を含む行を探して表示するコマンドラインツールを作成します。

## 実行方法
```bash
cargo run <検索文字列> <ファイルパス>
# 例: cargo run to poem.txt
```

## 実装ステップ

1. `src/main.rs` を開き、`TODO` コメントに従ってコードを完成させてください。
2. コマンドライン引数を受け取る処理を実装します。
3. ファイルを読み込む処理を実装します。
4. 検索ロジックを実装します。
5. エラーハンドリングを適切に行います（ファイルがない場合など）。

## ヒント
- 引数の取得には `std::env::args()` が使えます。
- ファイル読み込みには `std::fs::read_to_string()` が便利です。
- 文字列の検索には `line.contains(query)` が使えます。

## Advanced Challenges (発展課題)
Level 1をクリアしたら、以下の実務的な改善に取り組んでください。

### 1. ライブラリの活用 (`clap`)
標準の `std::env::args` の代わりに、引数解析ライブラリ `clap` (derive feature) を導入してください。
- `--ignore-case` フラグの実装。
- ヘルプメッセージの自動生成。

### 2. エラー設計 (`thiserror` / `anyhow`)
- ライブラリとしてのエラー型を `thiserror` で定義してください。
- `main` 関数でのエラーハンドリングを `anyhow` で簡潔に記述してください。

### 3. ベンチマーク (`criterion`)
- 巨大なテキストファイルに対して検索を行い、処理速度を計測してください。
