# Phase 2: Specialized Tracks

Coreカリキュラム修了おめでとうございます。
ここからは、自身の興味やキャリアパスに合わせて専門分野を深掘りします。

## Track A: Web & Backend (`A_web_backend`)
大規模なウェブサービスを構築するための技術スタックを学びます。
- **Keywords**: Redis, Message Queue (NATS/RabbitMQ), GraphQL, gRPC, Microservices.

## Track B: Systems & Network (`B_systems`)
OSに近い低レイヤーや、高パフォーマンスが求められるインフラ基盤技術を学びます。
- **Keywords**: Multithreading, Lock-free, unsafe, Custom Allocators, eBPF.

## Track C: Performance (`C_performance`)
「推測するな、計測せよ」。極限までパフォーマンスを引き出すための最適化技術です。
- **Keywords**: Profiling (flamegraph), SIMD, Memory layout optimization.

## Track D: Data Engineering (`D_data`)
大量のデータを効率的に処理するためのデータパイプライン技術です。
- **Keywords**: Apache Arrow, Polars, Batch Processing, Streaming.
